## ⚠️ 検討・軽微な指摘

### 1. 除外キーワードにスペースを含む場合

**対応済み**: スペースを含む場合は `-"${e}"` で囲むよう修正。単語の場合は `-${e}` のまま。

### 2. 設計書との差分

設計書 4.3 では「キーワード・除外・ユーザー」が常時展開。実装では以下に変更済み:
- 除外: 初期非表示、追加ボタンで表示
- ユーザー: 初期畳み

→ ユーザー要件による変更のため問題なし。設計書の更新は任意。

### 3. README の URL 表記

`https://hrstsh.com/tools/x-search-builder` に末尾スラッシュなし。他ツールも同様の表記。trailingSlash: always のため実際の URL は `/` 付き。README の簡略表記として許容範囲。
